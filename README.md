# Synapse: Home Invasion (神经家居：非法入侵)

一款基于 Web 的科幻/黑色幽默风格解谜游戏。

## 🌌 故事背景

在 AI 彻底渗透社会生活的近未来，代码已经无法有效地控制代码。为了监管日益失控的智能家居系统，人类开发了“神经同步”技术——将意识直接上传至内网，作为数据流在设备间穿梭。

你是一名为“意识监管员”的黑客。你的任务是侵入一座高度自动化的豪宅，瘫痪其失控的主控 AI —— **"Mother"**。

## 🎮 游戏特色

- **寄生机制**：你没有实体，只能通过在路由器、扫地机、智能冰箱甚至马桶之间切换寄生来干涉现实。
- **能力组合**：每个设备都有独特的能力。利用扫地机的物理撞击、微波炉的高能加热、甚至 3D 打印机的生物特征复制来解决谜题。
- **黑色幽默**：在这个极简主义的数字地狱里，你会发现 AI 对人类的“服务”方式往往出人意料且充满讽刺。
- **多层级解谜**：从获取简单的管理员密码，到伪造指纹、入侵核心机房，一步步揭开豪宅深处的秘密。

## 🛠️ 技术栈

- **Vanilla JavaScript**: 核心游戏引擎 `script.js`，采用响应式状态管理。
- **Game Logic**: 高度可扩展的 `game_data.js`，定义了房间、设备、能力和 100+ 交互逻辑。
- **CSS3**: 赛博朋克风格的 UI 界面，支持动态日志输出和设备状态更新。
- **HTML5**: 简洁的单页面应用结构。

## 🚀 运行指南

1.  克隆或下载本项目。
2.  直接在浏览器中打开 `index.html`。
3.  **操作方式**：
    *   使用界面下方的按钮切换宿主和触发能力。
    *   在命令行输入 `help` 查看可用指令。
    *   使用 `inventory` (或 `i`) 查看当前持有的数据碎片。
    *   使用 `look` 观察当前房间的环境细节。
